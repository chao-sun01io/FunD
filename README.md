FunD (Fun & Fund Data) is a site for tracking and displaying interested fund. 

It is a hobby project that demostrate  how to develop a website with 
modern web technologies and best practices. The goal is to provide a platform where users can monitor various funds, view detailed information, and analyze data trends in an interactive and user-friendly way.

This project is also intended as a learning resource, showcasing how to integrate Django with Redis and Celery for efficient background task processing and caching. The setup is containerized using Docker Compose, making it easy to develop, test, and deploy across different environments.
